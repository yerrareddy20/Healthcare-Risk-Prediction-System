# Healthcare Risk Prediction System

Developed by: Yerra Reddy Keerthi

## Project Overview
This project focuses on predicting healthcare risks using Machine Learning techniques.  
It helps in identifying high-risk patients based on clinical data, enabling early diagnosis and better medical decision-making.

## Goal
To support early detection of high-risk patients and improve healthcare outcomes through predictive analytics.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest

## Features
- Data Cleaning and Preprocessing
- Handling Missing Values
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Model Evaluation (Accuracy, Precision, Recall, F1-score)
